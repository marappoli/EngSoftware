Description

CSU02 - Criação de Perfil Atores Mestre e Jogador Sumário Criar um perfil no Sistema Pré-Condições A Roll Your Table exibe a página de perfil requisitando dados pessoais e preferenciais Pós-Condições A Roll Your Table armazena os dados Ponto de Inclusão Não há Ponto de Inclusão Ponto de Exclusão Não há Ponto de Exclusão Fluxo Principal: 1 - Cliente acessa o sistema. 2 - Roll Your Table exibe a página de perfil. 3 - Cliente preenche os dados requisitados. 4 - Roll Your Table armazena os dados recebidos para realizar uma filtragem. 5 - Cliente recebe uma lista de Jogadores/Mestres de acordo com a suas preferências. Fluxo Alternativo: 5.1 - Cliente deseja alterar seus dados pessoais ou preferências. 5.2 - Retorna ao passo 3 do Fluxo Principal. 5.3 - Altera os dados desejados. 5.4 - Retorna ao passo 4 do Fluxo Principal. Fluxo de Exceção: 3.1 - Cliente não preenche os dados corretamente. 3.2 - Roll Your Table informa que possuem dados preenchidos incorretamente. 3.3 - Retorna ao passo 2 do Fluxo Principal.
